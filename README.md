# Hospital Management System (English)

A basic hospital management system built as a university project.  
The goal was to simulate core hospital operations and practice structuring a larger Java application.

## Overview

This project models a simplified hospital system with functionality such as:
- managing patients
- handling doctor interactions
- organizing records
- basic system workflow logic

## Tech Stack

- Java
- Object-Oriented Programming (OOP)

## Features

- patient management
- doctor interaction logic
- structured system workflow
- basic data handling

## Context

This project was created during my university studies as part of learning Java and object-oriented programming.

## What I Learned

- working with multiple classes and system components
- applying OOP principles in practice
- organizing logic for real-world-like systems

## What I Would Improve Now

# Hospital Management System (Ukrainian)
Проєкт створено для управління внутрішньою системою лікарні та являє собою веб-застосунок, написаний мовою Java з використанням сервлетів та JSP.
База даних зберігається у файлі HospitalManagementSystem.sql, який необхідно імпортувати у свою СУБД.

Щоб запустити застосунок, необхідно запустити плагін tomcat7:run, після чого у будь-якому браузері ввести localhost:8080.
Після запуску застосунку з’явиться сторінка авторизації. Ось дані для входу в систему у ролі:

1) Адміністратора: login = Admin123, password = 123;
2) Секретаря на reception: login = Receptionist123, password = 123;
3) Лікаря: login = Doctor123, password = 123;

У кожного користувача різні можливості доступу та керування даними, тому варто увійти в систему під кожною з ролей.
